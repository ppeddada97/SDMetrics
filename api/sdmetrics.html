sdmetrics.evaluate(metadata, real, synthetic)[source]¶

This generates a MetricsReport for the given metadata and tables with the default/built-in metrics.

Parameters

  • metadata (sdv.Metadata) – The Metadata object from SDV.

  • real_tables (dict) – A dictionary mapping table names to dataframes.

  • synthetic_tables (dict) – A dictionary mapping table names to dataframes.

Returns

A report containing the default metrics.

Return type

MetricsReport
