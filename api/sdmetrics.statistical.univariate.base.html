static metric(real_column, synthetic_column)[source]¶

This function is expected to perform a statistical test on the two samples and return a tuple containing (value, goal, unit, domain). See the Metric object for what these fields represent.

Parameters

  • real_column (np.ndarray) – The values from the real database.

  • synthetic_column (np.ndarray) – The values from the synthetic database.

Returns

A tuple containing (value, goal, unit, domain) which corresponds to the fields in a Metric object.

(str, Goal, str, tuple)

metrics(metadata, real_tables, synthetic_tables)[source]¶

This function iterates over all the columns in all the tables and, if the data type of a column matches the data types for which this metric is defined, it computes the metric for that column and yields it.

Parameters

  • metadata (sdv.Metadata) – The Metadata object from SDV.

  • real_tables (dict) – A dictionary mapping table names to dataframes.

  • synthetic_tables (dict) – A dictionary mapping table names to dataframes.
