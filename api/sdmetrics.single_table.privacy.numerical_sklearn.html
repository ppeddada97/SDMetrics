class sdmetrics.single_table.privacy.numerical_sklearn.NumericalSklearnAttacker[source]¶

Bases: sdmetrics.single_table.privacy.base.PrivacyAttackerModel

Base class for numerical attacker based on sklearn models.

It is used to train a model to predict sensitive attributes from key attributes using the synthetic data. Then, evaluate the privacy of the model by trying to predict the sensitive attributes of the real data.
